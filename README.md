# My Chat App

My Chat App is a real-time messaging application built with **Flutter** and powered by **Firebase**. It enables users to send and receive text as well as media files in a fast, secure, and seamless environment. The app uses Firebase Authentication, Cloud Firestore, and Firebase Storage to ensure smooth communication and safe media handling.
